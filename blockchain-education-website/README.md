# Blockchain Education Website

An interactive, modern, premium‑looking educational website that teaches blockchain and Web3 concepts from beginner to intermediate level.

## Features
- Dark cyber‑punk UI with glassmorphism cards and animated background.
- Step‑by‑step lessons covering:
  - What is Blockchain? (Web1, Web2, Web3 comparison)
  - Centralized vs Decentralized systems
  - Bitcoin, block creation, Proof‑of‑Work
  - Ethereum, Proof‑of‑Stake
  - Public, Private, Hybrid blockchains
  - Transaction logs: UTXO & Account‑based models
  - Stablecoins, market overview, and logs
- Interactive simulators:
  - Mining/PoW demo, PoS validator demo
  - Network fault‑tolerance visualizer
  - UTXO calculator & Account balance viewer
  - Stablecoin mint/burn dashboard
- Adaptive quiz with downloadable certificate.

## Local Development
1. Clone the repo or download the source.
2. Open `index.html` in a browser (no server required for static assets).
3. For development, you can run a simple HTTP server:
   ```bash
   npx -y http-server ./ -p 8080
   ```
   Then navigate to `http://localhost:8080`.

## Deployment
The site is static and can be hosted on GitHub Pages, Netlify, Vercel, or any static web host.
